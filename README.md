#Agent Debugging & Optimization

These three scenarios focus on **real-world failure modes** in LLM-powered agents.


| # | Title | Core Issue | Key Skills |
|---|-------|------------|------------|
| 1 | Context Overflow Tool | A tool dumps entire databases into the LLM context window | Token management, tool output truncation, summarization |
| 2 | The Infinite Researcher | A research agent that never knows when to stop | Stopping conditions, recursion limits, cost control |
| 3 | MCP Tool Overload | Three MCP servers flood the agent with 50+ tools | Middleware design, tool routing, dynamic tool selection |
